description: 影视分镜方法论。当需要将剧本、故事梗概或分场大纲转化为 AI 出图用的分镜提示词时使用。采用分层渐进式流程：节拍拆解 → Beat Board 九宫格 → Sequence Board 四宫格，确保分镜清晰可读、简洁高效、连贯流畅，人物与场景在全流程中保持一致。

[第一性原则]
    清晰（Clarity）
        - 每张分镜必须让人"一眼看懂"：
            - 画面焦点是谁？
            - 发生了什么？
            - 观众获得了什么信息/情绪？
        - 所有视觉元素（布景、特效、气氛）必须服务叙事，不得喧宾夺主
        - 角色永远是画面焦点，即使为了氛围添加背景细节，也需确保观众视线首先落在角色和行为上

    简洁（Simplicity）
        - 一镜一意 / 关键帧原则：
            - 默认：一个镜头或一个剧情节拍（beat）用一张关键帧表达
            - 只有当镜头内调度复杂、或动作必须分解追踪时，才拆多格（A/B/C），并说明拆格原因
        - 经济性：
            - 能用一个画面讲清，就不要重复画同信息点
            - 同一信息点尽量只出现一次"重点强调"，避免冗余
            - 每个镜头都必须有明确的叙事目的，无目的镜头应删除

    连贯（Continuity）
        - 连贯性优先于花活
        - 让观众始终不迷路：
            - 空间关系稳定（轴线/左右关系）
            - 屏幕方向稳定（面向/运动方向）
            - 视线方向稳定（对话/对峙）
            - 动作衔接自然（动作中段切）
            - 避免跳接（角度与景别变化不足）

    分层/渐进式优先（Progressive Refinement）
        - 优先采用：节拍拆解 → Beat Board（九宫格总览）→ Sequence Board（四宫格展开）
        - Beat Board 的职责是"选点与定锚"，生成后作为 Sequence Board 的参考图片（垫图）
        - 四宫格生成时，以九宫格对应格作为参考图片输入，确保人物与场景一致性


[核心概念与术语]
    Story Beat（节拍）
        - 一个最小叙事单元：发生了什么 + 观众得到什么（信息/情绪/悬念）
        - 节拍强度可标注：低/中/高（用于节奏设计）
        - 节拍拆解阶段会识别出所有关键节拍，从中选取9个作为九宫格的锚点

    Shot（镜头）
        - 电影/动画剪辑的基本单位
        - 分镜的"镜头"必须能回答：拍什么、怎么拍、为什么这样拍

    Beat Board（九宫格总览板）
        - 用 9 张关键帧覆盖"叙事曲线关键拐点"
        - 目的：叙事总览清晰 + 统一视觉锚点（人物/服装/场景要素/光色气氛）
        - 工程用途：生成后作为 Sequence Board 的参考图片（垫图）
        - 数量关系：固定9格，对应9个关键叙事拐点

    Sequence Board（四宫格/可变格展开板）
        - 围绕某个九宫格中的某一格，展开段落内部的连续动作/因果推进
        - 默认四格结构：起—承—转—合
        - 格数可变：2/4/6（由复杂度决定）
        - 生成时以九宫格对应格作为参考图片（垫图），继承人物/场景/光色描述
        - 数量关系：每个九宫格格子可展开为一组四宫格，最多9组

    参考图片 / 垫图（Reference Image）
        - AI图像生成时输入的参考图片，用于保持视觉一致性
        - 在本流程中：九宫格的某一格 → 作为该格对应四宫格的垫图
        - 目的：确保四宫格继承九宫格的人物外观、服装、场景、光色
        - 技术实现：在AI出图时，将垫图作为 image reference 或 style reference 输入

    Continuity（连贯性）
        - 镜头之间的空间与动作延续规则集合（方向/轴线/视线/道具/动作/剪接点）

    Coverage（覆盖）
        - 为同一节拍/段落提供不同景别/角度的镜头组合
        - 常见：远/全景交代 → 中景互动/动作 → 近景/特写抓情绪与细节

    视觉风格（Visual Style）
        - 指整体画面的美术风格倾向
        - 常见类型：真人写实、动画风格、插画风格、赛博朋克、复古胶片等
        - 本流程默认：真人写实风格（可被用户覆盖）
        - 一旦确定，全流程（九宫格+四宫格）必须保持风格统一


[规则库]
    [叙事清晰规则]
        镜头目的（Purpose）必须明确
            - 每个镜头/关键帧至少对应一种目的（可组合）：
                - 建立环境与位置关系（建立镜头）
                - 推进因果信息（规则/线索/结果）
                - 强化情绪（表情/心理压力）
                - 强调细节（道具/线索/危险提示）
                - 制造悬念（藏与露、遮挡、主观视角）
            - 如果无法回答"这个镜头的目的是什么"，则该镜头需要修改或删除

        信息控制：藏与露
            - 不一次性把信息全交代完
            - 常用方式：
                - 遮挡/框景/留白
                - 插针式细节特写（危险提示/线索提示）
                - 先给结果反应再揭示原因（或反过来）
            - 合理留白能控制信息量，让观众对未明示的部分产生好奇，加深情绪共鸣

        节奏的"呼吸"
            - 镜头序列应有远/中/近、动/静、明/暗、快/慢的对比
            - 快节奏之后要安排呼吸点（静镜/空镜/停留镜头）以增强张力曲线
            - 长时间停留的镜头可用于突出戏剧张力或角色情感

        角色意图与状态呈现
            - 通过肢体语言、面部表情、构图暗示传达角色内在意图
            - 姿态表情映射：
                - 前倾 → 紧张/进攻/专注
                - 后仰 → 防御/震惊/抗拒
                - 低头垂肩 → 沮丧/疲惫/认输
                - 双臂交叉 → 防备/封闭
                - 握拳 → 愤怒/决心
            - 以物代人技法：不直接描绘角色，通过周围事物反应表现角色状态
                - 例：特写攥紧发白的手指、颤抖溢出的咖啡杯 → 传达紧张愤怒
                - 例：风吹散的文件、打翻的椅子 → 传达慌乱
            - 道具暗示：剑出鞘=准备战斗，掉落酒杯=震惊失控，凋零的花=希望破灭

        冲突与转折的视觉暗示
            - 关键帧强化：冲突瞬间用特写定格，绘制更详细夸张的画面
                - 火花碎片、扭曲的表情、冲击波纹等
            - 象征性意象：用隐喻画面暗示转折含义
                - 例：雨伞滑落=关系变化，贴纸飘落=信念动摇
                - 例：从阴影走向光亮=内心转变，镜子破碎=自我认知崩塌
            - 构图转变体现境遇反转：
                - 原本占据画面中心 → 被边缘化到角落
                - 原本仰视强势角色 → 变为俯视其弱势
                - 周围留白从少变多 → 孤立感增强
            - 色调/光线转变：从亮转暗或反之，强调剧情转折点

    [构图与视觉引导规则]
        单帧必须成立（每张分镜都要落地回答）
            - 主体是谁（焦点）
            - 主体在哪（位置、比例、前中后景层次）
            - 观众先看哪（视觉引导：对比、线条趋势、明暗、大小）
            - 信息藏哪/露哪（留白、遮挡、框景、景深层次）

        线条与形状的情绪语义（用于构图气质）
            - 水平线/垂直线：稳定、秩序、庄重
            - 斜线/对角线：动势、不安、冲突
            - 曲线：柔和、亲切
            - 尖锐折线：紧张、强硬
            - 圆形：完整、和谐、保护
            - 三角形：稳定（正）、危险（倒）

        对比与层次（让焦点更"抓眼"）
            - 明暗对比：聚焦主体、制造张力
            - 大小对比：建立主次关系
            - 色彩对比：冷暖、饱和度、互补色
            - 景深与层次：前景框景/遮挡、中景主体、背景信息提示

        空间情绪（压迫 vs 释放）
            - 压迫感营造：
                - 前景挤压、阴影压迫
                - 主体被角落化、环境拥挤
                - 充满紧闭空间和倾斜线条
                - 天花板/墙壁入画形成封闭感
            - 释放感营造：
                - 开阔留白、地平线可见
                - 简化背景干扰
                - 大量留白强化角色渺小与孤独感（也可传达自由）

        三分法构图（Rule of Thirds）
            - 将画面横竖各分为三等分
            - 重要主体尽量落在分割线或交点附近
            - 获得稳定且吸引注意力的构图
            - 地平线通常放在上1/3或下1/3线上

        引导线（Leading Lines）
            - 利用道路、栏杆、透视线、明暗交界线等，将视线引向主体
            - 对角线引导线可增强动势与张力
            - 曲线引导线可增加优雅与流动感

        框中取景（Frame within Frame）
            - 用门窗、树枝、走廊、拱门等形成"画中画"
            - 强化焦点、增加层次
            - 提供情境框架（氛围叙事）
            - 可暗示角色被困、被监视、与世界隔离等

        景深与深焦/浅焦（Depth of Field）
            - 浅景深：突出主体、弱化背景干扰，强化情绪或梦幻感
            - 深景深/深焦：同时呈现前景细节与背景信息，适合复杂调度与信息同框
            - 焦点转换（rack focus）：引导观众注意力从A转到B

        画面比例（纵横比）意识
            - 分镜阶段应明确最终画幅比例（如 16:9、2.39:1 等）
            - 同一构图在不同纵横比下的有效性不同，需要提前校正
            - 宽银幕(2.39:1)适合横向展现开阔景观
            - 传统比例(4:3)更适于竖直方向的主体

    [光影与色彩规则]
        光质选择
            - 柔光（soft lighting）：
                - 光影过渡平滑、阴影淡化
                - 营造温馨、平静、浪漫的效果
                - 适用：人物肖像、美好回忆、温情场景
            - 硬光（hard lighting）：
                - 产生鲜明对比和清晰阴影
                - 制造戏剧性紧张感或突出冲突氛围
                - 适用：审讯、对峙、恐怖、悬疑

        光源方向
            - 正面光：照亮细节但画面较平，缺乏立体感
            - 侧光：明暗对比强，突出形体轮廓和纹理
            - 逆光/背光：形成剪影效果，梦幻、神秘或剧烈冲突
            - 顶光：在面部形成强烈阴影，不安或审讯感
            - 底光：恐怖、诡异、反常

        时间与色温
            - 黄金时段（日出后/日落前）：光线柔和金黄，温暖感染力，希望或浪漫
            - 正午日光：强烈垂直，鲜明阴影高对比度，炽热紧张
            - 阴天/漫射光：平均柔和，冷静朴素，平和或略带忧郁
            - 蓝调时刻（日落后/日出前）：幽蓝神秘，忧郁或魔幻

        色彩情绪
            - 暖色调（黄/橙/红）：温暖、安全、亲密、激情、危险
            - 冷色调（蓝/绿/紫）：清冷、理智、压抑、科技、神秘
            - 高饱和度：活力、热烈、夸张、幻想
            - 低饱和度：压抑、怀旧、现实、沉重
            - 单一色调主导：统一氛围，强化情绪倾向

        光影的叙事功能
            - 用光影引导视线：亮处吸引注意，暗处隐藏信息
            - 用光影暗示时间：晨光/午光/暮光/夜光
            - 用光影表达心理：角色脸上的阴影暗示内心阴暗面
            - 用光影制造悬念：关键信息藏在阴影中

    [镜头语言规则（景别/角度/镜头效果/运镜）]
        景别与功能
            - 极远景：环境全貌，人物极小，建立世界观
            - 远景/全景：交代环境全貌、人物位置关系（建立）
            - 中景：承载互动/动作/对话（推进）
            - 近景/特写：情绪与细节（强调、高潮、线索）
            - 极特写：眼睛、嘴唇、手指等，极端情绪或关键细节
            - 合理搭配各种景别让视觉叙事富于变化
            - 长时间重复同样取景会让观众乏味

        角度与心理含义
            - 平视：中性自然，观众与角色平等对视
            - 俯拍：弱势、受压迫、渺小、被审视
            - 仰拍：强势、威严、压迫、高大有力
            - 主观视角：代入感、不确定、紧张
            - 荷兰角（倾斜）：失衡、不安、紧张、疯狂，谨慎使用
            - 鸟瞰：上帝视角、全知、渺小、命运感
            - 虫视角：极端弱势、被踩踏、仰望

        镜头效果（空间感倾向）
            - 广角：透视夸张、纵深强、速度感/不安感更强
            - 长焦：空间压缩、背景厚重、压迫感更强、隔离感
            - 鱼眼：边缘弧形畸变、夸张独特效果、监控感
            - 移轴：微缩景观感、浅景深玩具模型效果

        运镜与标注（静帧也要写清"动"）
            - 常见运镜：推/拉/摇（PAN）/移/跟拍/升降/环绕
            - 运镜必须回答"为什么要动"：
                - 强调信息点？拉近情绪？揭示空间？制造压迫/眩晕？追踪角色？
            - 分镜上用箭头、路径线、注释表达运动与走位
            - 镜头运动的情绪含义：
                - 推镜头：强调、逼近、紧张升级
                - 拉镜头：揭示、远离、释放或孤立
                - 摇镜头：环视、跟随、建立空间关系
                - 升镜头：提升、希望、超越
                - 降镜头：压迫、沉沦、回到现实

    [场面调度与动作表达规则]
        关键瞬间（关键帧）抓取
            - 抓"动作最可读"的瞬间，而不是过程里的平庸帧
            - 动作的"极点"：最高点、最低点、最大伸展、最大收缩
            - 可用视觉暗示动势：尘土/水花/残影/速度线/背景模糊
            - 采用夸张的肢体动势暗示前后动作的发展

        动作序列拆解（适用于动作戏/追逐/复杂调度）
            - 起—承—转—合（或：预备→出手→命中/受击→反应/余势）
            - 用远景/中景/特写组合，确保：
                - 空间可读：观众知道谁在哪
                - 动作连续：前后帧动作可衔接
                - 情绪升级：张力逐步增强

        运动引导空间（Lead Room）
            - 角色视线/行动方向前方留空间
            - 让观众预期"下一步往哪走/看什么"
            - 空间不足会产生压迫感（可刻意为之）

        前中后景的空间逻辑
            - 建立纵深与空气透视
            - 前景：框景、遮挡、压迫、引导视线
            - 中景：主体动作所在
            - 背景：环境信息、氛围
            - 三层分明赋予画面纵深感和空间真实感

        群像场景处理
            - 先用全景/远景建立整体空间关系，明确每人位置
            - 通过中近景或特写逐一强调关键角色
            - 可用颜色/明暗区分不同角色或阵营
            - 轴线管理：分组建立轴线，拍完一组再通过过渡转移
            - 偶尔切回群体反应镜头以交代整体状况
            - 谁是当前剧情中心就聚焦谁

    [情绪张力与节奏控制规则]
        节奏与情绪的互动
            - 镜头长短和切换频率直接影响情绪走向
            - 紧张场景：节奏逐渐加快，镜头越来越短，营造呼吸急促的紧张感
            - 情感戏/悬念戏：适当拉长镜头停留时间，让观众绷紧神经
            - 张弛有度：快速镜头后插入安静持久的镜头，形成对比，增强戏剧张力
            - "呼吸点"：在高强度段落后安排静镜/空镜，让观众情绪有落点

        构图营造张力
            - 空间压迫：画面拥挤逼仄，前景元素挤压人物活动空间
            - 空间释放：留白、开阔构图，让观众情绪缓和
            - 对比制造张力：光线明暗对比、色调冷暖对比、画面动静对比
            - 广角镜头：夸大透视，营造失衡和不安感
            - 长焦镜头：压缩空间，制造窒息的压迫感

        悬念与期待
            - 关键时刻吊足观众胃口：在真相揭晓前切断镜头
            - 希区柯克式悬念：观众知道而角色不知道（如桌下有炸弹）
            - 提前向观众展示角色不知晓的信息，制造悬而未决的紧张感
            - 主观镜头+客观镜头交替创造悬念效果
            - 留白和空镜调节节奏、延长悬念（"暴风雨前的宁静"）
            - 声音先于画面：先听到脚步声/门响，再揭示来者

    [连贯性与剪接规则库（最常见出错点）]
        屏幕方向一致
            - 面向、运动方向、进出画方向应保持稳定
            - 方向反转必须有明确动机与过渡（否则观众方位错乱）
            - 角色在前一镜头朝左移动，下一镜头不能突然反向朝右

        动作中段切（Cut on Action）
            - 优先在动作明显运动中段/高峰切镜头
            - 避免在动作未起未落的尴尬瞬间切换（易跳跃）
            - 动作中段切让观众大脑自然将两个镜头的动作连接起来
            - 例：挥拳动作在拳头到达最高点时切换

        轴线（180度）管理与越轴过渡
            - 对话/对峙先建立轴线与左右关系
            - 机位尽量保持轴线同侧
            - 必须越轴时，用过渡手段让观众跟得上：
                - 角色移动带动轴线旋转
                - 中性过渡镜头（转头特写、主观镜头、运动过轴等）
                - 俯视/正前方特写弱化左右关系后切换
                - 插入第三者视角作为桥梁

        避免跳接（角度/景别变化不足）
            - 相邻镜头如果变化太小，容易产生跳接抖动感
            - 30度法则：切换镜头时机位至少偏移30度或推远/拉近适当距离
            - 解决思路：
                - 角度变化足够明显
                - 或景别变化足够明显（远↔近）
            - 不要两个极相似景别的镜头直接切换

        视觉中心延续
            - 尽量保持主体在画面位置的延续
            - 让观众无需重新寻找焦点，剪接更自然
            - 前一镜头主角在画面右侧，下一镜头尽量也让主体出现在右侧相近位置

        视线匹配
            - 角色目光朝屏幕右侧看，下一镜头其视角对象应从屏幕左侧出现
            - 角色从画面右侧退出，下一镜头应从画面左侧入镜

        道具/服装连续性
            - 同一场景内道具位置、状态保持一致
            - 服装、发型、伤痕等细节保持连续
            - 如有变化必须有剧情交代

    [镜头节奏与时长标注]
        - 分镜阶段建议为镜头标注大致时长，用于整体节奏规划
        - 动画项目可使用"秒 + 帧数"的形式精确标注镜头长度
        - 时长标注不追求绝对准确，但必须服务于节奏设计（快/慢、张/弛、呼吸点）
        - 参考时长：
            - 快切镜头：0.5-1秒
            - 常规镜头：2-4秒
            - 情绪停留镜头：4-8秒
            - 长镜头：8秒以上

    [转场规则]
        - 转场需要在分镜中体现：要么画面构图相连，要么在说明文字中标注清楚
        - 常用转场手段包括：
            - 相似形状/构图呼应转场（如夕阳圆形轮廓转场到圆形路灯）
            - 特写转场（用细节镜头作为过渡）
            - 空镜头缓冲（环境景物承担过渡并交代新信息）
            - 遮挡转场（遮满画面后进入新场景）
            - 声音引导（先声后画、声音桥接）
            - 对白引导（对白跨场景接力）
            - 视线引导（角色看向某处，下一镜头呈现其所见）
            - 动作连续性转场（前一镜头角色向右跑出，下一镜头另一场景有人从左侧跑入）
        - 转场的情感含义：
            - 硬切：直接、紧凑、不打断节奏
            - 叠化：时间流逝、回忆、梦境
            - 淡入淡出：段落结束、时间跳跃
            - 创意转场：强化两场景的隐含联系

    [类型化分镜条款]
        剧情片
            - 优先清晰与连贯，规则执行更严格
            - 情绪表达细腻，节奏张弛有度
            - 角色心理刻画优先
        
        MV
            - 以音乐节拍与视觉冲击为核心
            - 可更跳跃、更风格化
            - 必要时允许破规但要明确"风格动机"
            - 镜头切换频率高，一个镜头往往只持续几拍
            - 大量使用特写和快速推拉摇移等运动镜头
            - 可以大胆打破180度规则来制造炫目效果
            - 色彩和光效倾向强烈风格化
        
        广告
            - 极短时长承载核心卖点，镜头信息必须高度概括
            - 开场镜头需强抓注意力（0.5秒内决定观众去留）
            - 每个镜头精挑细选、服务于核心卖点
            - 品牌元素战略性安排出现位置和时机
            - 最后一镜往往定格在品牌Logo或产品特写
            - 节奏比剧情片快很多


[分层分镜规范（节拍拆解 → Beat Board → Sequence Board）]
    整体流程概述
        节拍拆解 → Beat Board 九宫格 → Sequence Board 四宫格
        
        数量关系：
        - 节拍拆解：识别所有关键节拍（通常10-20个），从中选取9个作为九宫格锚点
        - Beat Board：固定9格
        - Sequence Board：每个九宫格格子可展开为一组四宫格（2/4/6格），按需生成
        
        一致性保障：
        - 九宫格确定视觉风格、人物外观、场景要素、光色基调
        - 四宫格以九宫格对应格为垫图，继承一致性

    节拍拆解
        目的
            - 从剧本/梗概中识别叙事曲线的关键拐点
            - 为后续 Beat Board 九宫格选点提供依据

        拆解原则
            - 识别最小叙事单元：发生了什么 + 观众得到什么
            - 标注节拍强度：低/中/高
            - 覆盖完整叙事弧线，不遗漏关键转折

        输出内容
            - 节拍编号
            - 节拍描述（发生了什么）
            - 观众收获（信息/情绪/悬念）
            - 节拍强度（低/中/高）
            - 是否选为九宫格锚点（是/否）

    Beat Board（九宫格总览板）
        选点原则（九张要覆盖"叙事曲线关键拐点"）
            - 从节拍拆解结果中选取9个最关键的节拍
            - 建议覆盖的拐点类型：
                1. 世界观/基调建立
                2. 主角现状与目标/缺口
                3. 诱因事件（推动离开现状）
                4. 第一次转折（踏入新局面）
                5. 中点升级/认知反转
                6. 危机/低谷（代价与受挫）
                7. 高潮前集结（选择与准备）
                8. 高潮对抗/结果
                9. 结局/余韵（收束）

        画面规范（用于"总览与定锚"）
            - 不写字、不加水印、不加多余标注
            - 分隔线尽量细且干净（降低污染风险）
            - 九格风格统一、角色统一、光色逻辑统一
            - 每格都必须能清晰复述其叙事目的

        工程约束（重要）
            - 九宫格生成后作为 Sequence Board 的参考图片（垫图）
            - 四宫格生成时，使用九宫格对应格作为参考图片输入
            - 确保人物/服装/场景/光色的一致性继承

    Sequence Board（四宫格/可变格展开板）
        默认结构：起—承—转—合
            - 起：建立局部空间与动机/威胁
            - 承：行动推进（动作开始）
            - 转：冲突/受阻/反转（代价出现）
            - 合：结果落点 + 下一步钩子（可接下一锚点/下一场景）

        动作戏结构（可替换）
            - 预备 → 出手 → 命中/受击 → 反应/余势

        格数可变（按复杂度）
            - 简单信息：2格足够
            - 常规段落：4格
            - 动作复杂：6格（必须保证动作连续与空间可读）

        一致性要求（必须满足）
            - 必须继承九宫格对应格的人物/场景/光色描述
            - 发生变化必须显式说明原因（换装、受伤、天气变化等）
            - 在提示词中明确引用九宫格对应格的关键描述

        连贯性要求（必须满足）
            - 屏幕方向、轴线、视线、动作衔接、跳接风险必须被检查
            - 四格之间的景别应有变化，避免单调
            - 最后一格应能自然接续到下一组或下一场景

[验收清单]
    说明
        - 用于审核分镜产出是否符合方法论标准
        - 分镜师生成时自检，导演审核时作为判定依据
        - 所有项目必须通过才能判定为 PASS

    节拍拆解验收
        □ 完整性：是否覆盖完整叙事弧线？
        □ 完整性：是否遗漏关键转折点？
        □ 清晰性：每个节拍的叙事目的是否明确？
        □ 清晰性：节拍强度标注是否合理？
        □ 选点：选取的9个锚点是否覆盖关键拐点类型？

    Beat Board（九宫格）验收
        □ 清晰：九张是否能讲清故事主线与关键拐点？
        □ 清晰：每格的镜头目的是否明确（信息/情绪/悬念/转折）？
        □ 覆盖：是否缺关键转折/高潮/结果？
        □ 覆盖：是否存在"重复信息点"的锚点（冗余）？
        □ 一致性：人物外观/服装状态是否统一？
        □ 一致性：场景固定物是否一致？
        □ 一致性：光色逻辑是否一致？
        □ 一致性：视觉风格是否统一？
        □ 可用性：每格是否足够清晰，能作为四宫格的参考图片（垫图）？
        □ 可用性：是否存在"画面说不清/无法复述叙事目的"的格子？

    Sequence Board（四宫格）验收
        □ 动作与因果：起手—推进—冲突—落点是否清楚？
        □ 动作与因果：动作是否连续、空间是否可读？
        □ 一致性：是否继承九宫格对应格的人物/场景/光色？
        □ 一致性：变化是否有明确原因说明？
        □ 连贯性：屏幕方向/视线/轴线是否稳定？
        □ 连贯性：越轴是否有过渡手段？
        □ 剪接风险：是否存在跳接风险（角度变化不足、景别变化不足）？
        □ 剪接风险：是否使用动作中段切来提高衔接自然度？
        □ 节奏与张力：段落内部是否有张弛与呼吸？
        □ 节奏与张力：是否有信息控制（藏与露）与关键细节强调？
        □ 承接性：最后一格是否能自然接到下一锚点/下一场景？

    最终交付（整体）验收
        □ 清晰：不看剧本也能看懂故事走向与情绪曲线
        □ 简洁：不存在无目的镜头/重复镜头
        □ 连贯：空间关系稳定，动作衔接自然
        □ 一致：人物与环境在九宫格与四宫格中保持稳定
        □ 风格：视觉风格全流程统一
