# 变更记录模板

本模板用于记录 Product Spec 的迭代变更历史。

---

## 文件命名

`Product-Spec-CHANGELOG.md`

---

## 模板格式

```markdown
# 变更记录

## [v1.2] - YYYY-MM-DD
### 新增
- <新增的功能或内容>

### 修改
- <修改的功能或内容>

### 删除
- <删除的功能或内容>

---

## [v1.1] - YYYY-MM-DD
### 新增
- <新增的功能或内容>

---

## [v1.0] - YYYY-MM-DD
- 初始版本
```

---

## 记录规则

- **版本号递增**：每次迭代 +0.1（如 v1.0 → v1.1 → v1.2）
- **日期自动填充**：使用当天日期，格式 YYYY-MM-DD
- **变更描述**：根据对话内容自动生成，简明扼要
- **分类记录**：新增、修改、删除分开写，没有的分类不写
- **只记录实际改动**：没改的部分不记录

## 完整示例

以下是一个「视频提示词生成器」的变更记录示例，供参考：

```markdown
# 变更记录

## [v1.3] - 2025-12-08
### 新增
- 新增「AI 一键优化」按钮，放置在输入区底部，点击后自动优化用户输入的描述
- 新增「历史记录」面板，记录最近 10 条生成的提示词

### 修改
- 左侧输入区比例从 35% 改为 40%
- 「生成提示词」按钮文案改为「开始生成」
- 默认生成数量从 3 条改为 5 条

---

## [v1.2] - 2025-12-05
### 新增
- 新增「收藏」功能，用户可收藏喜欢的提示词
- 输出卡片新增「风格标签」显示

### 修改
- 高级选项面板改为默认展开

### 删除
- 移除「自动保存」功能（用户反馈不需要）

---

## [v1.1] - 2025-12-03
### 新增
- 新增「高级选项」折叠面板
- 新增镜头类型下拉选择（特写 / 中景 / 远景 / 跟踪镜头）
- 新增镜头运动下拉选择（推 / 拉 / 摇 / 移 / 固定）
- 新增 Analyze Images 能力，支持图生视频模式

---

## [v1.0] - 2025-12-01
- 初始版本
```

## 写作要点

1. **版本号**：从 v1.0 开始，每次迭代 +0.1，重大改版可以 +1.0
2. **日期格式**：统一用 YYYY-MM-DD，方便排序和查找
3. **变更描述**：
   - 动词开头（新增、修改、删除、移除、调整）
   - 说清楚改了什么、改成什么样
   - 如果有原因，简要说明（如「用户反馈不需要」）
4. **分类原则**：
   - 新增：之前没有的功能、控件、能力
   - 修改：改变了现有内容的行为、样式、参数
   - 删除：移除了之前有的功能
5. **颗粒度**：一条记录对应一个独立的变更点，不要把多个改动混在一起
6. **AI 能力变更**：如果新增或移除了 Google AI Studio 能力，必须单独记录
